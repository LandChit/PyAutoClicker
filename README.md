# PyAutoClicker

Simple auto clicker in python with GUI :)

![Image](https://i.imgur.com/GK0VzKl.png)

# Packages that are being used
* pyqutogui
* keyboard
* pyqt5

# Features
* Use Coordinates (x,y)
* Follow Mouse
* Random Location
